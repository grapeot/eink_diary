# 风格探索实验 — 2026-06-05

项目早期的视觉语言选型实验。目的是为视觉日记的画作确定一个统一的风格基调。

## 做了什么

读取作者当时的真实近况（最近邮件、待办、AI 编程会话），合成出一份"最近在干什么"的画像，再用 `image_generation_skill` 生成 5 个**立意不同**的版本，竖版 3:4，适配 13.3" E6 的 1200×1600：

1. `01_light_rivers.jpg` — 意象抽象：中心一点发散多条光河，多线程工作交汇。色块强，最适合 E6 六色。
2. `02_dashboard_poster.png` — 中文 ambient 海报"今日图景"，六板块。GPT-Image-2 中文渲染清晰，最接近"信息日记"。
3. `03_second_brain.jpg` — 赛博 second mind：黑底发光神经/电路头像。对比强。
4. `04_ink_wash_calm.jpg` — 禅意水墨大留白，与忙碌做反差。
5. `05_field_guide.jpg` — 复古博物图谱：把近况画成标本（机器人/书+笔/讲台/纸船/咖啡/红酒/钢笔/K线星座）。

## 为什么图片不在仓库里

这些图是基于作者**真实个人近况**生成的实验产物。虽然是抽象意象画、不含可读私人信息，但内容反映了真实活动。按公开仓库的隐私边界，实际图片文件被 `.gitignore` 屏蔽，只保留本说明。图片留在本地。

## 结论去向

选定的风格基调会固化进项目配置（`DIARY_STYLE`），不依赖本目录的文件长期存在。
