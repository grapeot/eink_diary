# 昨日八瞬间 — 第一次端到端"一天日记"（2026-06-05）

第一次用真实全天数据端到端跑通"一天八幅瞬间日记"。

## 怎么做的

1. 用 collector 拉昨天（2026-06-05，数据全）8 个两小时窗口（08–10 … 20–22）的真实素材。
2. 以**微信**为精确窗口时序信号（有逐条时间戳），AI sessions 作当天背景。
3. 每个窗口挑一个最有张力的真实瞬间，写一个**多样化**的 prompt（构图/视角/情绪/光线各不同），主角鸭哥（鸭子）。
4. 并行生成 8 张，1K + medium + 3:4 + gpt-image-2。
