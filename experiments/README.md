# 实验记录

风格与内容方向的探索。图片是基于真实个人近况生成的，按隐私边界 gitignore，只留各目录的 README 记录结论。

## 演进脉络

### 1. style_exploration_20260605 — 首轮风格
5 个不同立意。结论：**具体看得懂的赢，抽象的输**。保留 `02 今日图景海报`、`05 博物图鉴`；弃用 `01 光河`、`03 赛博头像`、`04 水墨`（任何一天都能套，什么都没说）。

### 2. style_round2_20260606 — 活动清单具象化（被否决的范式）
8 个方向（博物图鉴/等距小屋/桌面flatlay/天际线/海报/地图/邮票/棋盘），都基于今天真实活动。结论：**它们底层是同一个东西**——把活动列成 bullet 再换皮肤具象化。是工程解法，但做的是"汇报"不是"日记"，信息用户本来就有。**整个"活动清单"范式被否决。**

### 3. three_lenses_20260606 — 三个镜头（方向确立）
跳出活动清单，试三种"有用"的定义，各 3 张：
- **瞬间 moment**：放大一个具体细节（op 卡壳顿悟 / 和 AI 争论推翻重来 / 推开一幅谁都能套的水墨画）。**最强**——撑得起整幅、最像日记、最不可替代。
- **质地 texture**：抽象画一天的形状（碎片 / 返工 / 强度）。形式对，作氛围点缀；需全天数据才准。
- **慢变量 slowvar**：仪表读数（人机耦合 / 创造vs维护 / 注意力发散）。概念清晰但分量轻，适合嵌角落做跨天暗线。

**结论：以"瞬间"为主**，质地/慢变量点缀。

### 4. duck_moments_20260606 — 鸭哥主角（确立）
把"瞬间"的主角换成拟人鸭子（鸭哥）。结论：**鸭子主角成立且可爱**，跨幅辨识度和连续性远好于泛泛的人，彩色 E6 上讨喜。主角定为鸭哥（鸭子或中国中年男子）。
