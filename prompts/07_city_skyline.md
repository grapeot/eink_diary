# 模板 07 · 一日天际线

把今天画成一条城市天际线 / 连排建筑，每一栋建筑代表今天的一件真实活动，建筑的招牌或外形暗示是什么事。像一座"今日之城"。

## 适用

想要整体感、把一天看成一座由今天的事盖起来的城的日子。

## Prompt 模板

```
A stylized city skyline representing one day, vertical 3:4 — a row of distinct buildings, each building standing for one real activity from today. The shape, signage, or rooftop icon of each building hints at the activity:
{{ACTIVITIES}}  ← 每件事盖成一栋楼（如 调研=图书馆塔, 编程=带代码窗格的玻璃楼, 教学=带讲台屋顶的楼, 带娃=彩色游乐场小楼, 咖啡=带咖啡杯招牌的小店, 交易=带行情屏的金融楼）

Sky reflects time of day: {{TIME}} (sunrise / midday / dusk). Clean flat illustration, layered depth. Limited 6-color e-ink palette: black, warm red, golden yellow, blue, green on off-white. Each building must clearly map to a specific activity from today. Cohesive skyline, not crowded. No text needed (signage icons ok).
```
