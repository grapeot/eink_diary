# 模板 05 · 工作台俯拍 flat-lay

把今天画成一张俯视的桌面摆拍（knolling / flat-lay）：今天用到的"工具与物件"整齐排列在桌面上，每件物件对应一件真实活动。

## 适用

想要质感、写实又具体的日子。物件即活动，一目了然。

## Prompt 模板

```
A top-down flat-lay (knolling) illustration of a desk, vertical 3:4, objects neatly arranged and aligned on a clean surface. Each object stands for one real activity from today:
{{ACTIVITIES}}  ← 把每件事对应一件桌面物件（如 写作=钢笔+稿纸, 编程=笔记本电脑显示代码, 调研=摊开的文献+便签, 带娃=一只小玩具/儿童画, 咖啡=手冲壶+豆罐, 红酒=一只酒杯, 财务=计算器+账本）

Time of day: {{TIME}} — a cup or light hint can signal it. Tidy, satisfying, well-spaced composition, soft shadows. Limited 6-color e-ink palette: black, warm red, golden yellow, blue, green on off-white. The objects must specifically reflect today's activities; avoid a generic stock desk. No text needed.
```
